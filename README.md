# Smart Parking System 🚗

##  Overview
This project is an IoT-based Smart Parking System that allows real-time parking management using a web dashboard and Firebase backend.

## Features
- Real-time parking slot tracking
- Entry & Exit system
- Firebase database integration
- Web dashboard interface
- Responsive UI

##  Tech Stack
- HTML, CSS, JavaScript
- Firebase (Realtime Database, Hosting)

## Project Structure
/public → All frontend files  
firebase.json → Firebase configuration
